# Sistema Inteligente de Predição de Surtos de Dengue

📌 Descrição do Projeto

Este projeto tem como objetivo desenvolver um sistema inteligente capaz de prever períodos de maior incidência de surtos de dengue, com foco na cidade do Recife.

A solução utiliza análise de dados históricos e variáveis ambientais para identificar padrões e antecipar possíveis picos de casos, auxiliando órgãos de saúde na tomada de decisão e prevenção.
